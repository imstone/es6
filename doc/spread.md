
## 扩展运算符...（三个点）
大家在ES6的很多地方会看到三个点...，比如
``` javascripit
var a = [1,2,3,4,5]; 
console.log(...a);
// 1, 2, 3, 4, 5

```
我们看到其实扩展运算符就是对数组的展开


### 我们来看看扩展运算符的几种用法

1.合并数组
``` javascript
var more = [3, 4, 5];
[1, 2, ...more ];
// [1, 2, 3, 4, 5]  ;
// 扩展运算符 把 数组more 展开然后和数组拼接到一起
相当于
[1, 2].concat(more)

```

2.和结构赋值结合

``` javasciprt
[a, ...b] = [1, 2, 3, 4, 5, 6]
// a === [1]
// b  === [2, 3, 4, 5, 6]
// 这里要注意的是扩展运算符一定要放到最后一个，不能出现如下的代码
[a, ...b, c] 

```



3.作为函数参数



4.分割字符串

``` javascript

var list = [...'test']

// ["t", "e", "s", "t"]




```
